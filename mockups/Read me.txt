Admin page:
с помощью мыщи выбрать пост -- удалить  пользователя/ пост (кнопки сверху)
-- редактировать пост нажав на кнопку с карандашом на соответствующем посте

Create post:
@ - Имя пользователя. Справа дата. Не редактируемые, устанавливаются по умолчанию
# - теги через пробел(или без), тег начинается с #, другие символы кроме # пробела и _(внутри тега) не допускаются
пробелы внутри тега также нельзя испольовать
Примеры (#flowers #red), (#flowers#red#red_flowers)
OK - создан новый пост
Cancel - новый пост не будет создан

Error:
после текста Error: идет код ошибки и ее краткое описание
home -- на главную страницу

Post:
пример поста, редактирование доступно только если это твой пост или ты администратор(кнопка с караднашем)
@ - Имя пользователя. Справа дата. Не редактируемые, устанавливаются по умолчанию
# - теги 
следующие 2 полоски - краткое описание
далее текст поста
стрелочка вниз (птичка) -- развернуть полностью
сердечко -- лайк

снизу вариант шапки для авторизированного пользователя
